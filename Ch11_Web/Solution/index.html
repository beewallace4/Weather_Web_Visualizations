Summary: Latitude vs X

This project aims to analyze the correlations between latitude and weather conditions. The four weather conditions we will be analyzing are Max Temperature, Humidity, Wind Speed, and Cloudiness. The data for this project comes from OpenWeather API. We pulled data from over 500 cities all over the world to get diversified data.

Overall, latitude seems to have the highest correlation with Max Temperature. Each of the graphs have their own analysis which can be accessed by clicking on the visualization or by using the navigation bar. A limitation of this dataset is that there is less data from below the equator. To view the visualizations next to each other in a larger size, navigate to the Comparison page. The data used for this project can be accessed from the Data page on the navigation bar.
